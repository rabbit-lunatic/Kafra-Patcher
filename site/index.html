Instalação

Comece em Minutos

Quatro passos simples para ter seu patcher funcionando.

01

Download

Baixe a última release do Kafra Patcher para Windows.

Download
02

Extração

Extraia os arquivos na pasta do seu cliente Ragnarok Online.

extrair KPatcher.zip -> pasta_do_ro/
03

Configuração

Crie o arquivo kpatcher.yml com suas configurações.

window:
  title: Meu Servidor RO
  width: 780
  height: 580

web:
  index_url: https://meuservidor.com/patcher/
04

Execução

Execute o KPatcher.exe e comece a jogar!

Configuração

Simples e Flexível

Configure tudo através de um arquivo YAML intuitivo e bem documentado.

kpatcher.yml 
# ═══════════════════════════════════════════════════════════════
# CONFIGURAÇÃO DA JANELA
# ═══════════════════════════════════════════════════════════════
window:
  title: Meu Servidor RO
  width: 780
  height: 580
  resizable: false
  frameless: true
  border_radius: 20

# ═══════════════════════════════════════════════════════════════
# BOTÃO JOGAR
# ═══════════════════════════════════════════════════════════════
play:
  path: ragexe.exe
  arguments: ["1sak1"]
  exit_on_success: true

# ═══════════════════════════════════════════════════════════════
# CONFIGURAÇÃO WEB E PATCHES
# ═══════════════════════════════════════════════════════════════
web:
  index_url: https://meuservidor.com/patcher/
  preferred_patch_server: Principal
  
  patch_servers:
    - name: Principal
      plist_url: https://meuservidor.com/plist.txt
      patch_url: https://meuservidor.com/patches/

# ═══════════════════════════════════════════════════════════════
# CONFIGURAÇÃO DO DISCORD RPC
# ═══════════════════════════════════════════════════════════════
discord:
  client_id: "123456789012345678"
  large_image: logo
  large_text: "Venha jogar!"
